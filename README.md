
# 💰 AI Currency Agent
![Homepage](/assets/homepage.png)
A full-stack AI Agent built with **Gemini 3.5 Flash**, **LangChain**, and **Flask**. This agent uses natural language to fetch real-time conversion rates and perform calculations automatically.

## 🚀 Features
- **Agentic Reasoning:** Uses LLM tool-calling to decide when to fetch rates.
- **Real-time Data:** Integrated with the Frankfurter API for up-to-date currency factors.
- **Clean UI:** Simple, responsive interface built with HTML5/CSS3.

## 🛠️ Tech Stack
- **Backend:** Python, Flask, LangChain
- **AI Model:** Google Gemini 3.5 Flash
- **Frontend:** JavaScript (Fetch API), CSS3

## ⚙️ Setup
1. Clone the repo: `git clone <https://github.com/himanshi20004/AI--Currency-Convertor-Agent.git>`
2. Create a virtual environment: `python -m venv venv`
3. Install dependencies: `pip install -r requirement.txt`
4. Add your `GOOGLE_API_KEY` to a `.env` file.
5. Run the app: `python app.py`
